### 思考・気づきの要約

- 技術や知識を身につける学習では、耳コピや教材による学習を積み上げ、DTM・作曲の全体像へ広げていく計画がある。
- 「楽にすること」は不要な負担を減らすための改善や自動化であり、「楽しむこと」は避けられない試行錯誤の中に意味や手応えを見つけることだと整理している。両者は異なるが、改善を続けながら困難も楽しめることが成長につながる。
- 人間関係では、相手との関係を一方的に評価せず、相互理解、誠実さ、会話における話題の重さやタイミングを意識したいと考えている。努力がそのまま成功確率の上昇につながらない関係では、確実性を求めすぎず試行回数を保てる姿勢が重要だと捉えている。
- 会話については、相手に何を尋ねるか、どの程度話に入り込むか、どのタイミングで反応するかが以前より分かるようになり、会話をリズムとして捉えられるようになった。
- 自信は他人との比較や一時的な評価ではなく、自分の強みや成長を認識し、それを行動に移すことで育てるものだと整理している。自信と傲慢を区別し、不安なときに思い出す言葉や実践方法を持とうとしている。
- 親子関係では、子供が大人になったときに親の子育てを肯定できるかという考えに強く影響を受けている。親への感謝を、携帯の使い方を教える、料理をするなど具体的な支援と態度で伝え、親を支えられる存在になりたいと考えている。
- 絵については、パーツ内のコントラストを抑え、画面全体の明暗と領域比率でマットな印象を作ることを意識している。色遣いと質感を中心に練習し、将来のコミッションに向けたポートフォリオ制作を進める計画がある。
